# Skills

This directory contains local skills plus a small amount of related agent configuration.

## Included skills

- `gitkraken-cli`
  A Git workflow skill that prefers the `gk` GitKraken CLI for everyday repo work such as branching, commits, pull requests, changelogs, and work item management.
- `linkedin-jobs`
  A job research skill for searching LinkedIn listings and fetching specific postings, mainly for job discovery, comparison, and resume-tailoring workflows.
